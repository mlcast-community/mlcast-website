ARCO-Ready Data Access

We leverage ARCO (Analysis-Ready, Cloud-Optimized) data patterns. By using Zarr and xarray, we ensure datasets are code-addressable, allowing you to stream terabytes of radar data directly into your training pipeline without local downloads.

Standardized

All datasets are curated into unified schemas, ensuring consistent coordinate systems and metadata compliant with CF-conventions.

EWC Hosted

Stored on the European Weather Cloud (EWC), minimizing latency and egress costs for high-throughput machine learning research.

Open & Accessible

Fully open-source catalog. Contribution is encouraged via PRs, making it a living community repository fornowcasting data.

Access in 3 lines of code

# Install the catalog interface

pip install mlcast-datasets

# Load data instantly

import mlcast_datasets
cat = mlcast_datasets.open_catalog()
ds = cat.precipitation.radklim_5_minutes.to_dask()

The MLCast Catalog acts as a single point of entry for all curated weather datasets. It leverages intake to map cloud-hosted Zarr stores into your local Python environment.
